# employee-management-systems

bout The Project

Managing employee leave requests manually can be tedious and prone to errors. This Employee Leave Management System automates the entire process, allowing employees to apply for leaves, check their balance, and track status, while managers can approve or reject requests with a single click.
 Key Features

     Role-Based Authentication: Separate dashboards for Employees, Managers, and Admin

     Leave Application: Employees can apply for different leave types (Casual, Sick, Earned).

     Approval Workflow: Managers receive real-time notifications to Approve or Reject leaves with comments.

     Leave Balance Tracker: Automatically calculates remaining leave days based on leave policy.

    Leave History & Status: Employees can track pending, approved, or rejected applications.

 Tech Stack

    

    Backend: Node.js / Express.js

    Database: MongoDB (Mongoose ODM)

    Authentication: JSON Web Tokens (JWT) & bcrypt.js
